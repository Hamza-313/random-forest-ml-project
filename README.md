## 📖 About the Project
This repository contains a **machine learning project** using the **Random Forest algorithm**.  
The notebook demonstrates:
- Loading and preparing a dataset  
- Splitting into training/testing sets  
- Building a Random Forest model with `scikit-learn`  
- Evaluating model performance  

---

## 📂 Dataset
The dataset used in this project was downloaded from Kaggle:  
👉 [Loan Data on Kaggle](https://www.kaggle.com/datasets/itssuru/loan-data)  

➡️ Please download the dataset manually and place the file (e.g. `loan_data.csv`) in the project root folder before running the notebook.
